# Beispielprojekt

Ein erstes Projekt für die Praktika im Fach Software Engineering.

## Inhalt

Es dient als Spielwiese für die Bearbeitung der Aufgaben, um die Grundlagen der
Versionsverwaltung mit Git, die Dokumentation mit AsciiDoc und den Umgang mit
GitHub kennenzulernen.

- **Git**, **GitHub** und **AsciiDoc**
